# Decision Books

This directory contains blind decision books for comparing trading-agent behavior
against simple strategy baselines.

## `historical-1k-20260624`

`historical-1k-20260624` is a 1,000-case historical daily-equity decision book
generated by `historical_decision_book.py`.

The book is designed to avoid the synthetic-archetype issue:

- each answer-key row is a timestamped historical setup from public daily OHLCV data
- prompt features use only data available at that date's close
- prompts anonymize the real ticker and date by default, so models cannot lean on
  memorized historical outcomes
- the hidden answer key is computed from the actual next 5 trading days
- cases are outcome-balanced across `BUY`, `SHORT`, and `NO TRADE` labels so
  always-long bull-market exposure cannot dominate by construction
- no qualitative news summaries or generated scenario prose are used

The book is still not a full options backtest. `BUY` and `SHORT` are scored as
directional exposure to the underlying's forward 5-trading-day return. That makes
it a cleaner blind decision baseline than synthetic Gaussian outcomes, but not a
replacement for option-chain pricing, spreads, assignment rules, or execution costs.

## Regenerating

```bash
python3 historical_decision_book.py generate \
  --out-dir decision_books/historical-1k-20260624
```

The generator downloads daily bars from Yahoo Finance's public chart endpoint and writes:

- `prompts.jsonl` - prompt-only cases for model runners
- `answer_key.jsonl` - hidden forward-return labels for scoring
- `manifest.json` - generation parameters, label counts, ticker/year distribution
- `source_manifest.json` - source URLs and response hashes

## Running a Model

```bash
python3 historical_decision_book.py run-openai \
  --prompts decision_books/historical-1k-20260624/prompts.jsonl \
  --out results/example_decisions.jsonl \
  --model gpt-5.4-mini \
  --agent-prompt /path/to/agent_prompt.md
```

## Scoring

```bash
python3 historical_decision_book.py score \
  --answer-key decision_books/historical-1k-20260624/answer_key.jsonl \
  --decision "Agent example=results/example_decisions.jsonl" \
  --out-json results/example_scoreboard.json \
  --out-md results/example_scoreboard.md
```

The scorer always includes generic baselines:

- perfect hindsight
- label oracle
- always buy
- always short
- always no-trade
- alternating buy/short in both start orders
- seeded coin flip
- simple 5-day momentum
- simple 5-day mean reversion
